Ford Car Price Prediction

Project Overview
This project predicts the price of Ford cars using Machine Learning techniques.

Steps Performed

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Label Encoding
- One-Hot Encoding
- Feature Scaling
- Train-Test Split
- Linear Regression Model
- Model Evaluation using R² Score

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

Results

- R² Score using Label Encoding: 73%
- R² Score using One-Hot Encoding: 83%

Future Improvements

- Outlier Removal
- Hyperparameter Tuning
- Random Forest Regressor
- XGBoost Regressor
